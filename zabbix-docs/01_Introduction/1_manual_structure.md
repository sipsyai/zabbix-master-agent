# 1 Manual structure

#### Structure

The content of this manual is divided into sections and subsections to provide easy access to particular subjects of interest.

When you navigate to respective sections, make sure that you expand section folders to reveal full content of what is included in subsections and individual pages.

Cross-linking between pages of related content is provided as much as possible to make sure that relevant information is not missed by the users.

#### Sections

[Introduction](about) provides general information about current Zabbix software. Reading this section should equip you with some good reasons to choose Zabbix.

[Zabbix definitions](/documentation/current/en/manual/definitions) explain the terminology used in Zabbix, while [Zabbix processes](/documentation/current/en/manual/concepts) provides details on Zabbix components.

[Installation](/documentation/current/en/manual/installation/getting_zabbix) and [Quickstart](/documentation/current/en/manual/quickstart/item) sections should help you to get started with Zabbix. [Zabbix appliance](/documentation/current/en/manual/appliance) is an alternative for getting a quick taster of what it is like to use Zabbix.

[Configuration](/documentation/current/en/manual/config) is one of the largest and more important sections in this manual. It contains loads of essential advice about how to set up Zabbix to monitor your environment, from setting up hosts to getting essential data to viewing data to configuring notifications and remote commands to be executed in case of problems.

[Service monitoring](/documentation/current/en/manual/it_services) details how to use Zabbix for a high-level overview of your monitoring environment.

[Web monitoring](/documentation/current/en/manual/web_monitoring) should help you learn how to monitor the availability of web sites.

[Virtual machine monitoring](/documentation/current/en/manual/vm_monitoring) presents a how-to for configuring VMware environment monitoring.

[Maintenance](/documentation/current/en/manual/maintenance), [Regular expressions](/documentation/current/en/manual/regular_expressions), [Problem acknowledgment](/documentation/current/en/manual/acknowledgment) and [Configuration export/import](/documentation/current/en/manual/xml_export_import) are further sections that reveal how to use these various aspects of Zabbix software.

[Discovery](/documentation/current/en/manual/discovery) contains instructions for setting up automatic discovery of network devices, active agents, file systems, network interfaces, etc.

[Distributed monitoring](/documentation/current/en/manual/distributed_monitoring) deals with the possibilities of using Zabbix in larger and more complex environments.

[Encryption](/documentation/current/en/manual/encryption) helps explaining the possibilities of encrypting communications between Zabbix components.

[Web interface](/documentation/current/en/manual/web_interface) contains information specific for using the web interface of Zabbix.

[Best practices](/documentation/current/en/manual/best_practices) contain useful tips for optimal and secure user experience.

[API](/documentation/current/en/manual/api) section presents details of working with Zabbix API.

[Extensions](/documentation/current/en/manual/extensions) is a collection of ways for extending Zabbix.

Detailed lists of technical information are included in [Appendixes](/documentation/current/en/manual/appendix).

[Quick reference guides](/documentation/current/en/manual/guides) contain step-by-step guides for accomplishing tasks with Zabbix.
