# 4 Favorite graphs

#### Overview

This widget contains shortcuts to the most needed graphs, sorted alphabetically.

The list of shortcuts is populated when you view a graph in Monitoring -> Latest data -> Graphs, and then click on its ![](/documentation/current/assets/en/manual/web_interface/frontend_sections/monitoring/button_add_fav.png) _Add to favorites_ button.

![](/documentation/current/assets/en/manual/web_interface/frontend_sections/monitoring/favorite_graphs.png)

All configuration parameters are [common](/documentation/current/en/manual/web_interface/frontend_sections/dashboards/widgets#common-parameters) for all widgets.
