# Method reference

This section provides an overview of the functions provided by the Zabbix API and will help you find your way around the available classes and methods.

## Monitoring

The Zabbix API allows you to access history and other data gathered during monitoring.

#### Dashboards

Manage dashboards and make scheduled reports based on them.

[Dashboard API](/documentation/current/en/manual/api/reference/dashboard) | [Template dashboard API](/documentation/current/en/manual/api/reference/templatedashboard) | [Report API](/documentation/current/en/manual/api/reference/report)

#### High availability cluster

Retrieve a list of server nodes and their status.

[High availability cluster API](/documentation/current/en/manual/api/reference/hanode)

#### History

Retrieve historical values gathered by Zabbix monitoring processes for presentation or further processing.

[History API](/documentation/current/en/manual/api/reference/history)

#### Trends

Retrieve trend values calculated by Zabbix server for presentation or further processing.

[Trend API](/documentation/current/en/manual/api/reference/trend)

#### Events

Retrieve events generated by triggers, network discovery and other Zabbix systems for more flexible situation management or third-party tool integration.

[Event API](/documentation/current/en/manual/api/reference/event)

#### Problems

Retrieve problems according to the given parameters.

[Problem API](/documentation/current/en/manual/api/reference/problem)

#### Maps

Configure maps to create detailed dynamic representations of your IT infrastructure.

[Map API](/documentation/current/en/manual/api/reference/map)

#### Tasks

Interact with Zabbix server task manager, creating tasks and retrieving response.

[Task API](/documentation/current/en/manual/api/reference/task)

## Services

The Zabbix API allows you to access data gathered during service monitoring.

#### Service Level Agreement

Define Service Level Objectives (SLO), retrieve detailed Service Level Indicators (SLI) information about service performance.

[SLA API](/documentation/current/en/manual/api/reference/sla)

#### Services

Manage services for service-level monitoring and retrieve detailed SLA information about any service.

[Service API](/documentation/current/en/manual/api/reference/service)

## Data collection

The Zabbix API allows you to manage the configuration of your monitoring system.

#### Hosts and host groups

Manage host groups, hosts and everything related to them, including host interfaces, host macros and maintenance periods.

[Host API](/documentation/current/en/manual/api/reference/host) | [Host group API](/documentation/current/en/manual/api/reference/hostgroup) | [Host interface API](/documentation/current/en/manual/api/reference/hostinterface) | [User macro API](/documentation/current/en/manual/api/reference/usermacro) | [Value map API](/documentation/current/en/manual/api/reference/valuemap) | [Maintenance API](/documentation/current/en/manual/api/reference/maintenance)

#### Items

Define items to monitor.

[Item API](/documentation/current/en/manual/api/reference/item)

#### Triggers

Configure triggers to notify you about problems in your system. Manage trigger dependencies.

[Trigger API](/documentation/current/en/manual/api/reference/trigger)

#### Graphs

Edit graphs or separate graph items for better presentation of the gathered data.

[Graph API](/documentation/current/en/manual/api/reference/graph) | [Graph item API](/documentation/current/en/manual/api/reference/graphitem)

#### Templates and template groups

Manage templates and link them to hosts or other templates.

[Template API](/documentation/current/en/manual/api/reference/template) | [Template group API](/documentation/current/en/manual/api/reference/templategroup) | [Value map API](/documentation/current/en/manual/api/reference/valuemap)

#### Low-level discovery

Configure low-level discovery rules as well as item, trigger and graph prototypes to monitor dynamic entities.

[LLD rule API](/documentation/current/en/manual/api/reference/discoveryrule) | [Item prototype API](/documentation/current/en/manual/api/reference/itemprototype) | [Trigger prototype API](/documentation/current/en/manual/api/reference/triggerprototype) | [Graph prototype API](/documentation/current/en/manual/api/reference/graphprototype) | [Host prototype API](/documentation/current/en/manual/api/reference/hostprototype)

#### Event correlation

Create custom event correlation rules.

[Correlation API](/documentation/current/en/manual/api/reference/correlation)

#### Network discovery

Manage network-level discovery rules to automatically find and monitor new hosts. Gain full access to information about discovered services and hosts.

[Discovery rule API](/documentation/current/en/manual/api/reference/drule) | [Discovery check API](/documentation/current/en/manual/api/reference/dcheck) | [Discovered host API](/documentation/current/en/manual/api/reference/dhost) | [Discovered service API](/documentation/current/en/manual/api/reference/dservice)

#### Export and import

Export and import Zabbix configuration data for configuration backups, migration or large-scale configuration updates.

[Configuration API](/documentation/current/en/manual/api/reference/configuration)

#### Web monitoring

Configure web scenarios to monitor your web applications and services.

[Web scenario API](/documentation/current/en/manual/api/reference/httptest)

## Alerts

The Zabbix API allows you to manage the actions and alerts of your monitoring system.

#### Actions and alerts

Define actions and operations to notify users about certain events or automatically execute remote commands. Gain access to information about generated alerts and their receivers.

[Action API](/documentation/current/en/manual/api/reference/action) | [Alert API](/documentation/current/en/manual/api/reference/alert)

#### Media types

Configure media types and multiple ways users will receive alerts.

[Media type API](/documentation/current/en/manual/api/reference/mediatype)

#### Scripts

Configure and execute scripts to help you with your daily tasks.

[Script API](/documentation/current/en/manual/api/reference/script)

## Users

The Zabbix API allows you to manage users of your monitoring system.

#### Users and user groups

Add users that will have access to Zabbix, assign them to user groups and grant permissions. Make roles for granular management of user rights.

[User API](/documentation/current/en/manual/api/reference/user) | [User group API](/documentation/current/en/manual/api/reference/usergroup) | [User directory API](/documentation/current/en/manual/api/reference/userdirectory) | [User role API](/documentation/current/en/manual/api/reference/role)

#### API Tokens

Manage authorization tokens.

[Token API](/documentation/current/en/manual/api/reference/token)

#### Authentication

Change authentication configuration options.

[Authentication API](/documentation/current/en/manual/api/reference/authentication)

## Administration

With the Zabbix API you can change administration settings of your monitoring system.

#### General

Change certain global configuration options.

[Autoregistration API](/documentation/current/en/manual/api/reference/autoregistration) | [Icon map API](/documentation/current/en/manual/api/reference/iconmap) | [Image API](/documentation/current/en/manual/api/reference/image) | [Settings API](/documentation/current/en/manual/api/reference/settings) | [Regular expression API](/documentation/current/en/manual/api/reference/regexp) | [Module API](/documentation/current/en/manual/api/reference/module) | [Connector API](/documentation/current/en/manual/api/reference/connector)

#### Audit log

Track configuration changes each user has done.

[Audit log API](/documentation/current/en/manual/api/reference/auditlog)

#### Housekeeping

Configure housekeeping.

[Housekeeping API](/documentation/current/en/manual/api/reference/housekeeping)

#### Proxies and proxy groups

Manage the proxies used in your distributed monitoring setup.

[Proxy API](/documentation/current/en/manual/api/reference/proxy) | [Proxy group API](/documentation/current/en/manual/api/reference/proxygroup)

#### Macros

Manage macros.

[User macro API](/documentation/current/en/manual/api/reference/usermacro)

### API information

Retrieve the version of the Zabbix API so that your application could use version-specific features.

[API info API](/documentation/current/en/manual/api/reference/apiinfo)
