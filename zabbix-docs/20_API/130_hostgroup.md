# Host group

This class is designed to work with host groups.

Object references:

  * [Host group](/documentation/current/en/manual/api/reference/hostgroup/object#host-group)

Available methods:

  * [hostgroup.create](/documentation/current/en/manual/api/reference/hostgroup/create) \- create new host groups
  * [hostgroup.delete](/documentation/current/en/manual/api/reference/hostgroup/delete) \- delete host groups
  * [hostgroup.get](/documentation/current/en/manual/api/reference/hostgroup/get) \- retrieve host groups
  * [hostgroup.massadd](/documentation/current/en/manual/api/reference/hostgroup/massadd) \- add related objects to host groups
  * [hostgroup.massremove](/documentation/current/en/manual/api/reference/hostgroup/massremove) \- remove related objects from host groups
  * [hostgroup.massupdate](/documentation/current/en/manual/api/reference/hostgroup/massupdate) \- replace or remove related objects from host groups
  * [hostgroup.propagate](/documentation/current/en/manual/api/reference/hostgroup/propagate) \- propagate permissions and tag filters to host groups' subgroups
  * [hostgroup.update](/documentation/current/en/manual/api/reference/hostgroup/update) \- update host groups
