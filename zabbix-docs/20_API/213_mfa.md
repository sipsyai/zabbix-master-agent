# MFA

This class is designed to work with MFA (Multi-Factor Authentication) methods.

Object references:

  * [MFA](/documentation/current/en/manual/api/reference/mfa/object#mfa)

Available methods:

  * [mfa.create](/documentation/current/en/manual/api/reference/mfa/create) \- create new MFA methods
  * [mfa.delete](/documentation/current/en/manual/api/reference/mfa/delete) \- delete MFA methods
  * [mfa.get](/documentation/current/en/manual/api/reference/mfa/get) \- retrieve MFA methods
  * [mfa.update](/documentation/current/en/manual/api/reference/mfa/update) \- update MFA methods
