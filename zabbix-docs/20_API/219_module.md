# Module

This class is designed to work with frontend modules.

Object references:

  * [Module](/documentation/current/en/manual/api/reference/module/object#module)

Available methods:

  * [module.create](/documentation/current/en/manual/api/reference/module/create) \- install new modules
  * [module.delete](/documentation/current/en/manual/api/reference/module/delete) \- uninstall modules
  * [module.get](/documentation/current/en/manual/api/reference/module/get) \- retrieve modules
  * [module.update](/documentation/current/en/manual/api/reference/module/update) \- update modules
