# Configuration

This class is designed to export and import Zabbix configuration data.

Available methods:

  * [configuration.export](/documentation/current/en/manual/api/reference/configuration/export) \- export configuration data
  * [configuration.import](/documentation/current/en/manual/api/reference/configuration/import) \- import configuration data
  * [configuration.importcompare](/documentation/current/en/manual/api/reference/configuration/importcompare) \- compare import file with current system elements
