# script.get

### Description

`integer/array script.get(object parameters)`

The method allows to retrieve scripts according to the given parameters.

This method is available to users of any type. Permissions to call the method can be revoked in user role settings. See [User roles](/documentation/current/en/manual/web_interface/frontend_sections/users/user_roles) for more information.

### Parameters

`(object)` Parameters defining the desired output.

The method supports the following parameters.

groupids | ID/array | Return only scripts that can be run on the given host groups.  
---|---|---  
hostids | ID/array | Return only scripts that can be run on the given hosts.  
scriptids | ID/array | Return only scripts with the given IDs.  
usrgrpids | ID/array | Return only scripts that can be run by users in the given user groups.  
selectHostGroups | query | Return a [`hostgroups`](/documentation/current/en/manual/api/reference/hostgroup/object) property with host groups that the script can be run on.  
selectHosts | query | Return a [`hosts`](/documentation/current/en/manual/api/reference/host/object) property with hosts that the script can be run on.  
selectActions | query | Return an [`actions`](/documentation/current/en/manual/api/reference/action/object) property with actions that the script is associated with.  
sortfield | string/array | Sort the result by the given properties.  
  
Possible values: `scriptid`, `name`.  
countOutput | boolean | These parameters are described in the [reference commentary](/documentation/current/en/manual/api/reference_commentary#common-get-method-parameters).  
editable | boolean  
excludeSearch | boolean  
filter | object  
limit | integer  
output | query  
preservekeys | boolean  
search | object  
searchByAny | boolean  
searchWildcardsEnabled | boolean  
sortorder | string/array  
startSearch | boolean  
  
### Return values

`(integer/array)` Returns either:

  * an array of objects;
  * the count of retrieved objects, if the `countOutput` parameter has been used.

### Examples

#### Retrieve all scripts

Retrieve all configured scripts.

[Request](/documentation/current/en/manual/api#performing-requests):
    
    
    {
               "jsonrpc": "2.0",
               "method": "script.get",
               "params": {
                   "output": "extend"
               },
               "id": 1
           }
