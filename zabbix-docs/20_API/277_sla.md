# SLA

This class is designed to work with SLA (Service Level Agreement) objects used to estimate the performance of IT infrastructure and business services.

Object references:

  * [SLA](/documentation/current/en/manual/api/reference/sla/object#sla)
  * [SLA schedule](/documentation/current/en/manual/api/reference/sla/object#sla-schedule)
  * [SLA excluded downtime](/documentation/current/en/manual/api/reference/sla/object#sla-excluded-downtime)
  * [SLA service tag](/documentation/current/en/manual/api/reference/sla/object#sla-service-tag)

Available methods:

  * [sla.create](/documentation/current/en/manual/api/reference/sla/create) \- create new SLAs
  * [sla.delete](/documentation/current/en/manual/api/reference/sla/delete) \- delete SLAs
  * [sla.get](/documentation/current/en/manual/api/reference/sla/get) \- retrieve SLAs
  * [sla.getsli](/documentation/current/en/manual/api/reference/sla/getsli) \- retrieve Service Level Indicator (SLI) data for SLAs
  * [sla.update](/documentation/current/en/manual/api/reference/sla/update) \- update SLAs
