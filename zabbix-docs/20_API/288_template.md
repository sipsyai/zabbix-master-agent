# Template

This class is designed to work with templates.

Object references:

  * [Template](/documentation/current/en/manual/api/reference/template/object#template)
  * [Template tag](/documentation/current/en/manual/api/reference/template/object#template-tag)

Available methods:

  * [template.create](/documentation/current/en/manual/api/reference/template/create) \- create new templates
  * [template.delete](/documentation/current/en/manual/api/reference/template/delete) \- delete templates
  * [template.get](/documentation/current/en/manual/api/reference/template/get) \- retrieve templates
  * [template.massadd](/documentation/current/en/manual/api/reference/template/massadd) \- add related objects to templates
  * [template.massremove](/documentation/current/en/manual/api/reference/template/massremove) \- remove related objects from templates
  * [template.massupdate](/documentation/current/en/manual/api/reference/template/massupdate) \- replace or remove related objects from templates
  * [template.update](/documentation/current/en/manual/api/reference/template/update) \- update templates
