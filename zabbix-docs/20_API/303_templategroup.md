# Template group

This class is designed to work with template groups.

Object references:

  * [Template group](/documentation/current/en/manual/api/reference/templategroup/object#template-group)

Available methods:

  * [templategroup.create](/documentation/current/en/manual/api/reference/templategroup/create) \- create new template groups
  * [templategroup.delete](/documentation/current/en/manual/api/reference/templategroup/delete) \- delete template groups
  * [templategroup.get](/documentation/current/en/manual/api/reference/templategroup/get) \- retrieve template groups
  * [templategroup.massadd](/documentation/current/en/manual/api/reference/templategroup/massadd) \- add related objects to template groups
  * [templategroup.massremove](/documentation/current/en/manual/api/reference/templategroup/massremove) \- remove related objects from template groups
  * [templategroup.massupdate](/documentation/current/en/manual/api/reference/templategroup/massupdate) \- replace or remove related objects from template groups
  * [templategroup.propagate](/documentation/current/en/manual/api/reference/templategroup/propagate) \- propagate permissions to template groups' subgroups
  * [templategroup.update](/documentation/current/en/manual/api/reference/templategroup/update) \- update template groups
