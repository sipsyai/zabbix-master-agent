# User

This class is designed to work with users.

Object references:

  * [User](/documentation/current/en/manual/api/reference/user/object#user)
  * [Media](/documentation/current/en/manual/api/reference/user/object#media)

Available methods:

  * [user.checkauthentication](/documentation/current/en/manual/api/reference/user/checkauthentication) \- check and prolong user sessions
  * [user.create](/documentation/current/en/manual/api/reference/user/create) \- create new users
  * [user.delete](/documentation/current/en/manual/api/reference/user/delete) \- delete users
  * [user.get](/documentation/current/en/manual/api/reference/user/get) \- retrieve users
  * [user.login](/documentation/current/en/manual/api/reference/user/login) \- log in to the API
  * [user.logout](/documentation/current/en/manual/api/reference/user/logout) \- log out of the API
  * [user.provision](/documentation/current/en/manual/api/reference/user/provision) \- provision LDAP users
  * [user.resettotp](/documentation/current/en/manual/api/reference/user/resettotp) \- reset user TOTP secrets
  * [user.unblock](/documentation/current/en/manual/api/reference/user/unblock) \- unblock users
  * [user.update](/documentation/current/en/manual/api/reference/user/update) \- update users
