# User macro

This class is designed to work with host-level and global user macros.

Object references:

  * [Global macro](/documentation/current/en/manual/api/reference/usermacro/object#global-macro)
  * [Host macro](/documentation/current/en/manual/api/reference/usermacro/object#host-macro)
    * [Macro configuration](/documentation/current/en/manual/api/reference/usermacro/object#macro-configuration)

Available methods:

  * [usermacro.create](/documentation/current/en/manual/api/reference/usermacro/create) \- create new host macros
  * [usermacro.createglobal](/documentation/current/en/manual/api/reference/usermacro/createglobal) \- create new global macros
  * [usermacro.delete](/documentation/current/en/manual/api/reference/usermacro/delete) \- delete host macros
  * [usermacro.deleteglobal](/documentation/current/en/manual/api/reference/usermacro/deleteglobal) \- delete global macros
  * [usermacro.get](/documentation/current/en/manual/api/reference/usermacro/get) \- retrieve host and global macros
  * [usermacro.update](/documentation/current/en/manual/api/reference/usermacro/update) \- update host macros
  * [usermacro.updateglobal](/documentation/current/en/manual/api/reference/usermacro/updateglobal) \- update global macros
