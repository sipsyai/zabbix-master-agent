# Appendix 3. Changes in 7.4

## 7.4.3

#### dashboard

[ZBXNEXT-10143](https://support.zabbix.com/browse/ZBXNEXT-10143) Added new dashboard widget field `highlight_row` in [problems](/documentation/current/en/manual/api/reference/dashboard/widget_fields/problems) widget.  

#### task

[ZBX-26202](https://support.zabbix.com/browse/ZBX-26202) `task.create`: _Admin_ and _User_ type users may now create 'Execute now' tasks.
