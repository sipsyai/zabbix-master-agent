# Discovery rule

This class is designed to work with network discovery rules.

This API is meant to work with network discovery rules. For low-level discovery rules, see [LLD rule API](discoveryrule).

Object references:

  * [Discovery rule](/documentation/current/en/manual/api/reference/drule/object#discovery-rule)

Available methods:

  * [drule.create](/documentation/current/en/manual/api/reference/drule/create) \- create new network discovery rules
  * [drule.delete](/documentation/current/en/manual/api/reference/drule/delete) \- delete network discovery rules
  * [drule.get](/documentation/current/en/manual/api/reference/drule/get) \- retrieve network discovery rules
  * [drule.update](/documentation/current/en/manual/api/reference/drule/update) \- update network discovery rules
